# Heart Disease Prediction using Deep Learning

## Project Overview
This project uses a Deep Neural Network (DNN) to predict the likelihood of heart disease based on clinical features. We achieved **80%+ accuracy** using a optimized Diamond architecture.

## Tech Stack
* **Python** (TensorFlow, Keras)
* **Scikit-Learn** (RobustScaler)
* **Seaborn/Matplotlib** (EDA & Visualization)

## Key Insights
* **RobustScaler** was used to handle outliers in Cholesterol data.

## How to Run
1. Install dependencies: `pip install -r requirements.txt`
2. Run the main script: `python src/model_train.py`
